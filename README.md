# Vitality blood
The Blood Donation Application aims to create a user-friendly platform that facilitates blood donation activities. The application will connect donors with those in need of blood, promoting a seamless and efficient donation process.


# Technology Stack
- Better Auth (authentication or authorization)
- Node (expressJS frameword)


# Features
- Role based authintication
- this application have 3 role features (donor, volunteer, admin)

#### donor
- donor can create a blood donation request
- donor can edit his/her donation requst
- donor can update his profile

#### volunteer 
- he/she have same features but he/she have extra feature 
- volunteer can manage all donations requst

#### admin
- admin have all feares with user management like  user block and user role change and user status change
